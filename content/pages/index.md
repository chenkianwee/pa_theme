Project Automated envisions a computationally streamlined architectural design process where repetitive and laborious tasks are automated, designers are free to focus on what truly matters, solving the design challenges at hand and creating a better built environment for everyone. This computational design process will be available to anyone, anytime and anywhere because it uses only affordable, open and transparent technologies. No expensive licenses, subscriptions or vendor lock-ins. 

Project Automated pursuits this vision through independent research projects documented as open-source <a href="blogs/">articles</a>, <a href="research/01_software.html">programming codes</a> and <a href="education/01_ebooks.html">ebooks</a>.
